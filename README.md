# Guide

## nvm setup

## To check present working directory

```sh
pwd
```

## Go to home directory

```sh
cd ~ # cd /home/madan or cd 
```

## To run a js script

```sh
node app.js # node ./app.js
```

## Initialize a repo

```sh
git init
```

## To initialize a npm project

```sh
npm init -y
```

## Commit

```sh
git add . # Add files to staging area
git commit -m "Initial commit"
```
